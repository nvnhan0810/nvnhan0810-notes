## Cài đặt package 

```bash
npm i @notifee/react-native
```

## Setup cho android

Mở file `android/build.gradle` Thêm vào key `allprojects > repositories` nội dung 

```json 
 maven {
    url "$rootDir/../node_modules/@notifee/react-native/android/libs"
}
```

## Setup với IOS

Chạy lệnh 

```bash
cd ios && pod install
```

Về cách sử dụng thì tham khảo doc nha.
